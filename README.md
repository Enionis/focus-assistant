# Фокус Помощник - Мини-приложение для Max

Приложение для управления задачами с использованием техники Pomodoro. Работает как мини-приложение на платформе Max.

## Структура проекта

```
xaxaton_max/
├── webapp/          # Веб-приложение для Max
│   ├── index.html
│   ├── main.js
│   └── styles.css
│
└── bot/             # Серверная часть бота
    ├── server.js
    └── package.json
```

## Функциональность

- Создание и управление задачами
- Разбиение задач на подзадачи
- Pomodoro таймер
- Статистика и достижения
- Настройка активных часов для напоминаний
- Автоматические напоминания о незавершенных задачах
